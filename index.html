Introduction

Hi, there! You’ve discovered the most docile markdown editory on the web. I created this because I was frustrated with all the other options out there. These were either:

  1. Too memory intensive, or

You won’t find anything fancy in this domesticated environment, but you will be able to edit whatever you like. And, we won’t steal your data; It’s all yours!
